# Excel – Complete Analyst-Level Notes

This repository contains a **comprehensive guide to Microsoft Excel** for data analysts, students, and professionals. It covers fundamental and advanced topics, including formulas, functions, data analysis, visualization, dashboard creation, automation, and workflow best practices.

## Repository Contents

- `full notes.md` – Complete, structured Excel notes covering all essential topics:
  - Excel basics, workbook, and sheet structure
  - Keyboard shortcuts for efficiency
  - Data management and cleaning techniques
  - Cell referencing (relative, absolute, mixed)
  - Aggregation and counting functions
  - Tables and PivotTables
  - Charts and PivotCharts
  - Templates and dashboard creation
  - Data validation
  - Logical, lookup, text, date/time, and error handling functions
  - Power Query for ETL operations
  - Power Pivot and DAX for advanced analytics
  - Macros and automation
